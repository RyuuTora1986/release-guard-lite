How It Works

2

Classify Risk

4

Report + Alert

FAQ

Does it use AI to judge risk?

No. Risk classification uses deterministic rules: which files changed + whether checks passed. No LLM guessing, no hallucination risk.

What if I don't use Discord?

Discord push is optional. Leave the webhook blank and you still get the local Markdown report. Use --json to pipe results to any tool.

Can I use it in CI?

Yes. Exit code 0 = low risk, 1 = medium/high. Add one line to your GitHub Actions workflow. Use --json for machine-readable output.

Is it free?

Yes, fully open source under MIT license. Use it in personal and commercial projects.
