Release Guard Lite

A pre-release risk checker for small teams. Catch risky changes before they ship.

$ npx release-guard-lite init
Click to copy

Problems It Solves

🚨

Missed pre-release checks

Stop relying on memory. Automated checks run every time, catching what humans forget.

🤖

AI-generated code risks

AI writes fast but changes a lot. Get a risk summary before merging AI-generated PRs.

📋

No release audit trail

Every run produces a Markdown report. Shareable, searchable, and trackable.

⏰

Time spent babysitting deploys

Get Discord alerts with risk level. Know when to pay attention and when to relax.

How It Works

1

Read Diff

Compares your branch against the base branch using git

2

Run Checks

Executes your configured commands (lint, test, typecheck)

3

Classify Risk

Deterministic rules, not AI guessing. High-risk files + failed checks = HIGH

4

Report

Markdown report saved locally + Discord embed notification

Get Started in 30 Seconds

No signup. No config files to learn. Just init and run.
